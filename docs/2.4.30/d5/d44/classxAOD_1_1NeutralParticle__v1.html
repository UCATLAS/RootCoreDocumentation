Defining parameters functions

The 'defining parameters' are key to the concept of a NeutralParticle, and give the values for the IParticle interface ( pt(), phi(), eta() etc.). They use the Trk::NeutralPerigee coordinate system, and are defined as: . The parameters are expressed with respect to an origin (returned by vx(), vy() and vy() ), currently intended to be the 'beamspot'. This origin is expected to be the same for all track particles in a collection (and this may be be enforced).
