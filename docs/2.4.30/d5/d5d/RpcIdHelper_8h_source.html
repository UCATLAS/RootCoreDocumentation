00021 // Description
00022 // -----------
00023 // This factory class constructs RPC identifiers and ranges and provides access
00024 // to the levels.  ATLAS note ATL-MUON-2002-??? provides a complete description
00025 // of the hierarchical identifier scheme.  RpcIdHelper provides an interface to the
00026 // following fields of the identifier.
00027 //
00028 // Field           Range               Notes
00029 // ==============================================================================
00030 // StationName     unsigned integer    maps to "BIL","BMS",etc.
00031 // StationEta      [-8,8]              increases with Z
00032 // StationPhi      [1,8]               increases with phi
00033 // Technology      [2]                 maps to "RPC"
00034 // DoubletR        [1,2]               increases with R
00035 // DoubletZ        [1,3]               increases with |Z|
00036 // DoubletPhi      [1,2]               increases with phi
00037 // GasGap          [1,2]               increases with R
00038 // MeasuresPhi     [0,1]               0 if measures Z, 1 if measures phi
00039 // Strip           [1,n]               increases with Z for MeasuresPhi=0
00040 //                                     increases with phi for MeasuresPhi=1
00041 // ==============================================================================
00042 //
00043 // Inheritance
00044 // -----------
00045 // Inherits from MuonIdHelpers/MuonIdHelper.
00046 //
00047 // Author
00048 // ------
00049 // Steven Goldfarb <Steven.Goldfarb@cern.ch>
00050 //
00051 // Compact ID implementation by
00052 // Ketevi A. Assamagan <ketevi@bnl.gov>
00053 // BNL, March 11th, 2003
00054 //
00055 // ******************************************************************************
