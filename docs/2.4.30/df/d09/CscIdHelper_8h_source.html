00021 // Description
00022 // -----------
00023 // This factory class constructs CSC identifiers and ranges and provides access
00024 // to the levels.  ATLAS note ATL-MUON-2001-014 provides a complete description
00025 // of the hierarchical identifier scheme.  CscIdHelper provides an interface to the
00026 // following fields of the identifier.
00027 //
00028 // Field           Range               Notes
00029 // ==============================================================================
00030 // StationName     unsigned integer    maps to "CSS", "CSL", etc.
00031 // StationEta      [-1,1]              -1 for backward, 1 for forward endcap
00032 // StationPhi      [1,8]               increases with Phi
00033 // Technology      [1]                 maps to "CSC"
00034 // ChamberLayer    [1,2]               increases with |Z|
00035 // WireLayer       [1,4]               increases with |Z|
00036 // MeasuresPhi     [0,1]               0 if measures R, 1 if measures Phi
00037 // Strip           [1,n]               increases with R   for MeasuresPhi=0
00038 //                                     increases with Phi for MeasuresPhi=1
00039 // ==============================================================================
00040 //
00041 // Inheritance
00042 // -----------
00043 // Inherits from MuonIdHelpers/MuonIdHelper.
00044 //
00045 // Author
00046 // ------

00048 //
00049 // Compact Id implementation by
00050 // Ketevi A. Assamagan <ketevi@bnl.gov>
00051 // BNL, February 27th, 2003
00052 //
00053 // ******************************************************************************
