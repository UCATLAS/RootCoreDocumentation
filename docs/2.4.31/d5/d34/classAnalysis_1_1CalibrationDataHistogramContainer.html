Detailed Description

This is the class holding information for histogram-based calibration results.

int CalibrationDataHistogramContainer::getTagWeightAxis ( ) const
virtual 
Test whether this calibration object is one for "continuous" calibration

(this has some subtle consequences for the treatment of bin-to-bin correlations). The return value will be -1 in case this is not a "continuous" calibration object, and the axis number (0 for X, 1 for Y, 2 for Z) otherwise.
