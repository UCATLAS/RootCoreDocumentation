00018 // Description
00019 // -----------
00020 // This factory class constructs sTGC identifiers and ranges and provides access
00021 // to the levels.  ATLAS note ATL-MUON-2001-014 provides a complete description
00022 // of the hierarchical identifier scheme.  sTgcIdHelper provides an interface to the
00023 // following fields of the identifier.
00024 //
00025 // Field           Range               Notes
00026 // ==============================================================================
00027 // diehl: todo: update this section for sTGC
00028 // StationName     unsigned integer    maps to T1F,T3E,etc.
00029 // StationEta      [-4,-1]             backward endcap (-1 at lowest R)
00030 //                 [1,4]               forward endcap (1 at lowest R)
00031 // StationPhi      [1,8]               increases with phi
00032 // Technology      [4]                 maps to sTGC
00033 // GasGap          [1,2]               doublet: increases with |Z|
00034 //                 [1,3]               triplet: increases with |Z|
00035 // ChannelType     [0,2]               0 if pad, 1 if strip, 2 if wire
00036 // Channel         [1,n]               increases with R         for channelType=1 (strip)
00037 //                                     increases with phi       for channelType=2 (wire)
00038 //                                     increases with phi and R for channelType=0 (pad)
00039 //                                     
00040 // ==============================================================================
00041 //
00042 // Inheritance
00043 // -----------
00044 // Inherits from MuonIdHelpers/MuonIdHelper.
00045 //
