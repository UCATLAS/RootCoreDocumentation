16 // class sTgcIdHelper
17 // ******************************************************************************//
18 // Description
19 // -----------
20 // This factory class constructs sTGC identifiers and ranges and provides access
21 // to the levels. ATLAS note ATL-MUON-2001-014 provides a complete description
22 // of the hierarchical identifier scheme. sTgcIdHelper provides an interface to the
23 // following fields of the identifier.
24 //
25 // Field Range Notes
26 // ==============================================================================
27 // diehl: todo: update this section for sTGC
28 // StationName unsigned integer maps to T1F,T3E,etc.
29 // StationEta [-4,-1] backward endcap (-1 at lowest R)
30 // [1,4] forward endcap (1 at lowest R)
31 // StationPhi [1,8] increases with phi
32 // Technology [4] maps to sTGC
33 // GasGap [1,2] doublet: increases with |Z|
34 // [1,3] triplet: increases with |Z|
35 // ChannelType [0,2] 0 if pad, 1 if strip, 2 if wire
36 // Channel [1,n] increases with R for channelType=1 (strip)
37 // increases with phi for channelType=2 (wire)
38 // increases with phi and R for channelType=0 (pad)
39 //
40 // ==============================================================================
41 //
42 // Inheritance
43 // -----------
