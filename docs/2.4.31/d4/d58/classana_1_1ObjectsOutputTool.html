virtual StatusCode ana::ObjectsOutputTool::execute ( xAOD::TEvent *  outputStore)
virtual

Write all containers and their output aux containers to the provided persistent store. This method loops over the configured objects and configured suffixes, writing each out as basename_suffix. The user must still call TEvent::fill in order to save an event to the output file.

Set object suffixes via a list of systematics.

Add an already configured output tool. This method allows users to configure the individual output container tools as desired. User must fully initialize the IOutputToolOld before this. This tool will take ownership of the IOutputToolOld.
