Detailed Description

a IJetSelector implementation which select according to ratio of attributes

This class is typically usefull for ratios commonly used (like Tau3/Tau2) and which are stored as separate attributes. Same properties as above plus

  • "Attribute2" (string) the denominator attribute name
  • "AttributeType2" (string) the denominator attribute type
  • "VectorIndex" (int) the index in the denominator vector.
