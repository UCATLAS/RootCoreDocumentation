81 /* Set the Phi angle of a rotation matrix,
82  * leaving Theta and Psi unaltered.
83  *
84  * Actually it returns a new rotation matrix
85  * built with the new Phi angle, and with the
86  * Theta and Psi angles taken from the original matrix.
87  *
88  * N.B.
89  * if "convention = 0" --> "Z-X-Z" convention ==> DEFAULT!!
90  * if "convention = 1" --> "Z-Y-Z" convention
91  *
92  * N.B.!!
93  * for normal usage, use the default notation (simply leave it empty, or use convention=0),
94  * or, alternatively, be sure to use the same convention in both setPhi() and getPhiThetaPsi().
95  *
96  *
97  */
