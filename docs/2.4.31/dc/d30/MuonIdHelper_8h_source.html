32 // ******************************************************************************
33 // class MuonIdHelper
34 // ******************************************************************************
35 //
36 // Description
37 // -----------
38 // This is a base class for the Muon Identifier and Range factory classes.
39 // ATLAS note ATL-MUON-2001-014 provides a complete description of the
40 // hierarchical identifier scheme. MuonIdHelper provides an interface to the
41 // common station "prefix" of the identifier, as described below.
42 //
43 // Field Value Notes
44 // ==============================================================================
45 // StationName [0..n] maps to "BIL", "EMS", "T1F", etc.
46 // there can be holes in the list of indices
47 // StationEta integer increases with eta (0 at eta=0)
48 // StationPhi non-zero integer increases with phi
49 // Technology [0..5] maps to "MDT", "CSC", "RPC", "TGC", "STGC", "MM"
50 // ==============================================================================
51 //
52 // Inheritance
53 // -----------
54 // Inherits from DetectorDescription/AtlasDetectorID.
55 // It is inherited by the following technology-specific concrete classes.
56 //
57 // MdtIdHelper factory of MDT-specific Identifiers and Ranges
58 // CscIdHelper factory of CSC-specific Identifiers and Ranges
59 // RpcIdHelper factory of RPC-specific Identifiers and Ranges
60 // TgcIdHelper factory of TGC-specific Identifiers and Ranges
61 
62 // for nSW
63 // sTgcIdHelper factory of sTGC-specific Identifiers and Ranges
64 // MmIdHelper factory of MicroMegas-specific Identifiers and Ranges
65 
66 //
67 // Methods for External Use
68 // ------------------------
69 // int stationNameIndex(const std::string name) (returns stationName index)
70 // int technologyIndex(const std::string name) (returns technology index)
71 // std::string stationNameString(const int index) (returns stationName string)
72 // std::string technologyString(const int index) (returns technology string)
73 //
74 // Author
75 // ------
76 // Steven Goldfarb <Steven.Goldfarb@cern.ch>
77 //
78 // Compact Id implementation by
79 // Ketevi A. Assamagan <ketevi@bnl.gov>
80 // CERN, February 20th, 2003
81 //
82 
83 // for nSW
84 // Nektarios Chr. Benekos <nectarios.benekos@cern.ch>
85 // CERN, Feb. 2013
86 // ******************************************************************************
