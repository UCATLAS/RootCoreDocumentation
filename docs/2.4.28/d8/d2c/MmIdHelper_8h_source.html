00017 // Description
00018 // -----------
00019 // This factory class constructs MicroMegas (MM) identifiers and ranges and provides access
00020 // to the levels.  ATLAS note ATL-MUON-2001-014 provides a complete description
00021 // of the hierarchical identifier scheme.  
00022 // MmIdHelper provides an interface to the following fields of the identifier.
00023 //
00024 // Field           Range               Notes
00025 // ==============================================================================
00026 // nectar: todo: update this section for MM
00027 // StationName     unsigned integer    maps to T1F,T3E,etc.
00028 // StationEta      [-5,-1]             backward endcap (-1 at lowest R)
00029 //                 [1,5]               forward endcap (1 at lowest R)
00030 // StationPhi      [1,48]              endcap: increases with phi
00031 //                 [1,24]              forward: increases with phi
00032 // Technology      [5]                 maps to MM --> <label name="MM" value="5" /> in IdDictMuonSpectrometer_R.01.xml
00033 // Multilayer       [1,2]               barrel: increases with R
00034 //                                     endcap: increases with |Z|
00035 // GasGap          [1,2]               doublet: increases with |Z|
00036 //                 [1,3]               triplet: increases with |Z|
00037 // Channel         [1,n]               increases with R for IsStrip=0 (wire gang)
00038 //                                     increases with phi for IsStrip=1 (strip)
00039 // ==============================================================================
