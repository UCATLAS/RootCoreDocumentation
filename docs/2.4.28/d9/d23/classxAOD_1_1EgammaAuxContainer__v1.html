Detailed Description

Temporary container used until we have I/O for AuxStoreInternal

This class is meant to serve as a temporary way to provide an auxiliary store with Athena I/O capabilities for the egamma EDM. Will be exchanged for a generic auxiliary container type (AuxStoreInternal) later on.
