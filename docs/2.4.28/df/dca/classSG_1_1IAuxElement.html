Detailed Description

Flag that a class may have auxiliary data associated with it.

This is an empty class, existing only to flag that a class may have auxiliary data associated with it. SG::AuxElement derives from this, but one should use this class for tests rather than SG::AuxElement to avoid dependency issues.

This is an `interface' in the sense that it identifies a group of classes, but it defines no functionality itself. In fact, this class should not have a virtual table.
