Detailed Description

Implementation of tool to correct tau ID variables for pile-up in a generic way.

Input comes from 2 ROOT files with TF1 functions containing the mean variable value as a function of the chosen pile-up estimator (Nvtx offline, mu online)

The correspondence between the ROOT File curve names and the TauParameters is handled via a small structure defined in initialize
