00267 // These are the intermediate classes from which @c DataList derives
00268 // in the case of multiple or virtual derivation.  The actual
00269 // definitions are in the @c icc file.  They have the following
00270 // properties:
00271 //   - They derive from those of @c B1, @c B2, and @c B3 that are not
00272 //     @c NoBase.
00273 //   - They perform (static) checking to ensure that there's a unique
00274 //     base class in the hierarchy.
00275 //   - They make the following types available from the base classes:
00276 //     - @c PtrList
00277 //     - @c BaseContainer
00278 //     - @c size_type
00279 //     - @c difference_type
00280 //     - @c allocator_type
00281 //   - @c has_virtual is defined as @c true.
00282 //   - An override of @c dl_typeid is defined, to prevent ambiguities.
